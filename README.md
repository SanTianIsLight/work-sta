## 20230809 谈谈最近的感受

1. 情况很槽糕，因为已经两个月了，眼睑上那个点，一直没有消下去，而且发现自己的眼袋好像也加重了，翻看了以前的照片，确实是没有的，所以啊！随着年纪的增长，问题也会逐渐增加的。年纪大了也要懂事了，别人你关心不到，哈哈哈，那就只能关心自己了。

2. 心里的疙瘩：7.29广州之行，在珠江新城站那里，跟大姐，二姐和棋棋汇合后，我们出站在逛，准备去附近吃点东西。附近都是高楼大厦，都是办公楼，吃的地方很少，相当于一边找一边逛了。突然，迎面走来一个美女，第一反应就是很像高中的同学，就那么一瞬间吧，好像她笑了（不知道是不是自己想多了）。（若多看两眼，肯定会心动的）

3. 在一个两人都能看清对方的状态下（因为现在很多人都近视，或者看手机没有注意）。两个很久不见的同学，突然见到，那一瞬间，应该是很惊讶后然后很惊喜。若不是这样的，首先排除你们之前有什么很深的矛盾，那样见面是回避或者矛盾再次加深。然后就是其中有一个人很早就发现并确认你了，他早就惊讶过了。最后就是你们并不认识，初次见面而已。若是真的很久不见，见面的眼神应该都是彼此发光吧。特别有意思的就是，曾经你们有故事，相恋或暗恋。如果玩的好，也不至于将近十年不见吧。

4. 记一次遇见：那是刚毕业也就是2019年的国庆，在家里呆了5天后，跑到学校去见一些同学和朋友。其中在去万达回学校的路上，遇见一个人。一个我没有带眼睛就能确定的是很熟悉的人。但当我跟他打招呼的时候，他很懵，而我是那样的兴奋，因为这样碰上的概率太小了，但是遇上了。后来分析他看我的眼神，应该能够确定，他确实不认识我。但是他的样貌我肯定是认识的。同样的外貌下，身高和胖瘦都是差不多的。唯一不一样的就是他忘记了，我们一起上了四年的课，在深圳我们还见过面的（虽然没有过多的交流和玩耍，但是我们有四年的上课经历呀）。不，这肯定不是他。我弟我小郭，当时对我的表现很惊讶，我也极力的解释他是我大学同学。（曾有那么一瞬间，我都怀疑自己眼睛是不是有问题，出现幻觉了） **后来我跟另一个同学聊起的时候，才知道他有一个双胞胎弟弟。真相得以大白。**

5. 治好自己的眼睛是首要任务。关心好自己后，看能不能找个关心的人，最好她也关心你。然后就是技能的提升，英语阅读能力，代码编码能力等，这样才能提升自己所处的环境。现在这个阶段，不仅要垂直拓宽也要水平开发，更加完善自己。哈哈哈，要不要跟那个很像的高中同学，联络联络，聊聊最近的状况。
