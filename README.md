4. 评价一下预约会诊和直接医院看，这两种看病方式：都是有利有弊。
	|  | 预约会诊 | 医院直接看 |
	| --------- | ---------------------------------------- | ---------------------------------------- |
	| 利 | 充分合理的安排时间，方便大城市中的人，这些生活节奏比较快的 | 可以直接找医生看，不用因为人排满了，就改天看 |
	| 弊 | 需要根据医院状态和规则，安排自己的时间，以及排满了，要等很久 | 可能去的时候，医院人很多，导致需要排很久的队，不确定性比较大 |
> **这个评价完全是自己的看法，有评价不当的地方，你说的都是对的，都是合理的。**

进手术室，先平躺，然后闭眼，之后消毒，消完毒之后，慢慢的医生没有动静了，大概就这样过了二十分钟的样子吧，中途我只叫了医生，说手术怎么还不开始。在进手术室之前，我想着这件事是不是要跟亲人说一下，大小也是一个手术呀，后面想想还是算了吧。让我觉得，可能需要跟亲人说一下这个状况，是因为手术前，签了免责协议。后面没有跟家里说，是因为动的是眼睑上的那个肿块，关系应该不是很大吧。在闭着眼睛这二十分钟，主要还是想这个手术会发生什么，会不会因为疼，出现什么睁开眼睛。自己忍受不了疼而出现意外。然后就是手术后，自己眼睛能不能睁开和是不是接下来，是不是要休息一周了。当然还有一些孤独感。脑海中闪现过，强者就是孤独的。孤独感，主要还是来自于没有关心吧。最后就是有一丝慌张，慌张来自于手术室没有一点声音，当时也不能睁开，总以为那个医生在后面准备。
